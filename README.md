# Call Application

## Description
This is a simple Call Application built using JavaScript.  
It allows users to simulate calling functionality with a clean and user-friendly interface.

## Features
- Call button functionality
- Simple and clean UI
- Interactive JavaScript logic
- Beginner-friendly project

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Click on the call button to use the app

## Project Purpose
This project was created for learning JavaScript basics and practicing DOM manipulation.
